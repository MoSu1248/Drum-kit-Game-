🥁 JavaScript Drum Kit
A simple and interactive Drum Kit built using HTML, CSS, and JavaScript. You can use your keyboard to play different drum sounds, just like a real drum kit!

🚀 Features
Press keys on your keyboard to play drum sounds

Visual feedback when a key is pressed

Responsive and beginner-friendly codebase

🎹 How to Use
Open the index.html file in your browser.

Press the following keys to play sounds:

Key	Sound
A	Kick
S	Snare
D	Hi-Hat
F	Tom 1
G	Tom 2
H	Crash
J	Ride

(You can customize the keys and sounds in the JavaScript file.)

🛠️ Technologies Used
HTML5

CSS3

JavaScript (Vanilla)
